Do maxit iterations of RTISI-LA for a single frame

This function just creates a plan, executes it and destroys it.

Note the function can be run inplace i.e. frames and frames2 can point to the same memory location.

Parameters
[in]framesN frames M samples long
[in]gAnalysis window
[in]specg1Analysis window used in the first iteration for the newest lookahead frame
[in]specg2Analysis window used in the other iterations for the newest lookahead frame
[in]gdSynthesis window
[in]aHop size
[in]MFFT length, also length of all the windows (possibly zero-padded).
[in]NNumber of frames N = lookback + 1 + lookahead
[in]sTarget magnitude, N frames M samples long
[in]lookaheadNumber of lookahead frames
[in]maxitNumber of iterations
[out]frames2N output frames M samples long
