A VST plugin for blockwise, reassigned nonuniform filter banks

Authors: Zdeněk Průša, Nicki Holighaus

A plugin for audio visualization using nonuniform filter banks and the reassignment method is presented. The plugin provides a low delay moving spectrogram with respect to virtually any time-frequency filter bank by interfacing the C backend of the LTFAT open-source toolbox for time-frequency processing. Low delay is achieved by blockwise processing and complemented by an intuitive GUI, both implemented with the JUCE C++ Library. By computing three filter banks separately, the reassigned representation can be obtained with little additional computational effort.

Related functions in LTFAT

Since version 2.1.0, the LTFAT toolbox includes functions performing reassignment of nonuniform filterbank spectrograms.

filterbankphasegrad
Computes phase gradient using given filterbank.
filterbankreassign
Performs actual reassignment using phase gradient.
