Towards High Quality Real-Time Signal Reconstruction from STFT Magnitude

Zdeněk Průša
Abstract:

We present an extensive and reproducible comparison with the state-of-the-art algorithms.

The preprint is available here ltfatnote048.pdf.

Reproducible research:

The following archive ltfatnote048.zip contains scripts reproducing figures and tables from the paper.

Please note that LTFAT toolbox (version>=2.1.3, available here) and PHASERET (version>=0.2.0, available here) must be installed in order to run the scripts.

A real-time comparison of algorithms RTPGHI, RTISI-LA, GSRTISI-LA and the proposed one is available in demo_blockproc_phaseret2.m from the PHASERET toolbox.

Additional content:

The following table show errors in dB for all 70 samples from the EBU SQAM database for the proposed algorithm (RTPGHI(1) + GSRTISI-LA(1)) and for RTPGHI(1), RTISI-LA(2) and GSRTISI-LA(2). The number in the brackets denotes the number of look-ahead frames used. The number of per-frame iterations was fixed to 24.

The playback can be started by selecting one of the table cells. All the files were compressed by oggenc from vorbis-tools 1.4.0 with the default settings. Your browser must support HTML5 audio player and it must be able to decode ogg files. Alternativelly an archive containing all the files compressed using lossless flac encoder can be downloaded from here.
