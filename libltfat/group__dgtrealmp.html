◆ dgtrealmp_iterstep_callback

typedef int dgtrealmp_iterstep_callback(void *userdata, dgtrealmp_state *state, LTFAT_COMPLEX *c[])

Callback template to be called every iterstep iteration

See also
dgtrealmp_setparbuf_iterstep dgtrealmp_setparbuf_iterstepcallback
Parameters
[in/out]userdata User defined struct
[in]stateState struct
[in/out]cres Coefficient domain residual
[in/out]c Selected coefficients
[in]atomsNumber of selected atoms so far
[in]itersNumber of iterations performed so far
[in]errdbApproximation error estimate
Returns
Status code: =0 continue iterations, >0 stop, <0 stop with error

◆ dgtrealmp_parbuf_init()

int dgtrealmp_parbuf_init ( dgtrealmp_parbuf **  p)

Initialize the parameter buffer with default values and no dictionary

The default values are:

  • Target SNR: 40 dB
  • Kernel relative threshold: 1e-4
  • Phase convention: LTFAT_TIMEINV
  • Pedantic search: disabled
  • Max. number of iterations: 0.8*L
  • Max. number of atoms: 0.8*L
Parameters
[in]pDGTREALMP parameter buffer
