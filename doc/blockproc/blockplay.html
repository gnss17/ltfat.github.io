BLOCKPLAY - Schedules block to be played

Usage

blockplay(L)

Input parameters

f Samples.

Description

Function schedules samples in f to be played. Since playrec handles playing and recording in a single command, the actual relay of samples to playrec is done in the next call of blockread. In case no audio output is expected (in the rec only mode), the function does nothing.
