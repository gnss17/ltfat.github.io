Towards High Quality Real-Time Signal Reconstruction from STFT Magnitude

Zdeněk Průša
Abstract:

We present an extensive and reproducible comparison with the state-of-the-art algorithms.

Reproducible research:

The following archive ltfatnote048.zip contains scripts reproducing figures and tables from the paper.

Please note that LTFAT toolbox (version>=2.1.3, available here) and PHASERET (version>=0.2.0, available here) must be installed in order to run the scripts.

Additional content:

The following table show errors in dB for all 70 samples from the EBU SQAM database for the proposed algorithm and for RTPGHI(1), RTISI-LA(2) and GSRTISI-LA(2). The number in the brackets denotes the number of look-ahead frames used. The number of per-frame iterations was fixed to 12.

The playback can be started by selecting one of the table cells (the enabled ones turn yellow when the cursor hovers over them). All the files were compressed by oggenc from vorbis-tools 1.4.0 with the default settings. Your browser must support HTML5 audio player and it must be able to decode ogg files. Alternativelly, the file path is shown below the player and it can be downloaded by Save Link As ...
