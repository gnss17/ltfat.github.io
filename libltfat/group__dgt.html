◆ ltfat_phaseconvention

enum ltfat_phaseconvention

Discrete Gabor Transform Phase convention

There are two commonly used phase conventions.

Frequency invariant:

\[ c(m,n) = \sum_{l=0}^{L-1}\! f(l) \overline{g(l-na)} \me^{-\mi 2\pi l m/M } \, \]

This is commonly used by mathematicans. Effectivelly, each frequency channel is demodulated to the baseband.

Time invatiant:

\[ c(m,n) = \sum_{l=0}^{L-1}\! f(l) \overline{g(l-na)} \me^{-\mi 2\pi (l-na) m/M } \, \]

This is commonly used by engineers. This is equivalent to a filterbank.

See also
dgt_phaselock dgtreal_phaselock dgt_phaseunlock dgtreal_phaseunlock
