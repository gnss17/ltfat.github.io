FRAMETF2COEF - Convert coefficients from TF-plane format

Usage

cout=frametf2coef(F,cin);

Description

frametf2coef(F,cin) converts the frame coefficients from the time-frequency plane layout into the common column format.

Not all types of frames support this coefficient conversion. The supported types of frames are: 'dgt', 'dgtreal', 'dwilt', 'wmdct', 'ufilterbank', 'ufwt',`'uwfbt'` and 'uwpfbt'.
