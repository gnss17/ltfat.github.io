Help me!

If you have questions about LTFAT, then please send an email to the mailing list. You don't have to be a member of the mailing list to send a message, but a moderator must aprove your message (to avoid spam).

I want to report a bug

If you think you have found a bug in LTFAT, then please check out the bug tracker and put it in there. The bug tracker contains a list of all current issues with LTFAT and all feature request for the future.
