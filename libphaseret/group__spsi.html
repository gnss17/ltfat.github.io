Masked SPSI algorithm implementation

Works as spsi() except arg(c[ii])=phase[ii] if mask[ii] evaluates to true.

Parameters
[in]sM x N arrays, target magnitude
[in]aHop factor
[in]MNumber of frequency channels
[in]NNumber of time frames
[in]maskM x N array, mask of coefficients with known phase
[in]phaseM x N array, known phase
[in,out]initphaseLength M vector, [in] phase of -1 frame, [out] phase of [N-1] frame. Can be NULL, but then a memory allocation will occur.
[out]cM x N array of coefficients
