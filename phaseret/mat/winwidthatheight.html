WINWIDTHATHEIGHT - Window width at height

Usage

width = winwidthatheight(gnum, height)

Input parameters

gnum Window.
atheight Relative height.

Output parameters

width Window width in samples.

Description

winwidthatheight(gnum,atheight) computes width of a window gnum at the relative height atheight. gnum must be a numeric vector as returned from gabwin. If atheight is an array, width will have the same shape with correcpondng values.
