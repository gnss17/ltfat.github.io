Detailed Description

A simple data stream blocking interface employing a pair of circular buffers. It is intended for block-wise processing of audio streams or long audio files. The blocks might be overlapping.

The processor will invoke the registered callback function every hop samples and will provide a block of samples of length winLen.

The output data stream is delayed by procDelay samples behind the input stream.

Optionally, the blocks can be weighted by pre- and post- windows.
