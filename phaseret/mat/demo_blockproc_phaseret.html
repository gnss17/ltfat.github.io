DEMO_BLOCKPROC_PHASERET - STFT phase reconstruction algorithms comarison

Usage

demo_blockproc_phaseret('gspi.wav')
demo_blockproc_phaseret(...,'a',a)`

Description

For additional help call demo_blockproc_phaseret without arguments.

This file does real-time reconstruction from the STFT magnitude using 3 algorithms. No effect, just plain reconstruction.

demo_blockproc_phaseret(...,'a',a) allows setting the hop factor. Default value is 256.

This code produces the following output:

DEMO_BLOCKPROC_PHASERET:
To run the demo, use one of the following:

demo_blockproc_phaseret('gspi.wav') to play gspi.wav (any wav file will do).
demo_blockproc_phaseret('dialog') to choose the wav file via file chooser dialog GUI.
demo_blockproc_phaseret(f,'fs',fs) to play from a column vector f using sampling frequency fs.
demo_blockproc_phaseret('playrec') to record from a mic and play simultaneously.

Avalable input and output devices can be listed by |blockdevices|.
Particular device can be chosen by passing additional key-value pair 'devid',devid.
Output channels of the device cen be selected by additional key-value pair 'playch',[ch1,ch2].
Input channels of the device cen be selected by additional key-value pair 'recch',[ch1].
